## ðŸ›  Installation & Set Up

1. Clone the repo

   ```sh
   git clone https://github.com/zxffo/flow/
   ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   npm i
   ```

4. Start the development server

   ```sh
   npm start
   ```
